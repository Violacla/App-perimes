      • Nouvelles fonctionnalités :

      • 1. Le bouton " Notification " s'appelle désormais " Suivi Périmés " : 
        
                Il s'ouvre toujours de la même manière, mais contient maintenant 3 onglets : 
        
                - " Trimestre " => Affiche les produits qui périmeront dans les 3 prochains mois.

                - " À retirer " => Affiche les produits périmés le mois précédent, à retirer des rayons.
        
                Vous pouvez imprimer les tableaux avec le bouton imprimante.
